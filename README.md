# Middleware de Integración Biométrica

Aplicación desarrollada en C# para integrar dispositivos biométricos con una base de datos PostgreSQL, permitiendo la autenticación de usuarios y el registro de accesos de forma centralizada.

## Objetivo

Desarrollar una capa intermedia de software capaz de conectar un dispositivo biométrico con un sistema de gestión de usuarios, facilitando la validación de identidades y el almacenamiento de registros de acceso.

## Tecnologías Utilizadas

* C#
* .NET
* PostgreSQL
* SQL
* SDK Biométrico

## Funcionalidades

* Conexión con dispositivo biométrico.
* Validación de usuarios registrados.
* Registro automático de accesos.
* Almacenamiento de información en PostgreSQL.
* Consultas para recuperación y administración de registros.
* Integración entre hardware biométrico y base de datos.

## Arquitectura General

1. Captura de información biométrica mediante el dispositivo.
2. Procesamiento y validación de datos en la aplicación.
3. Consulta de usuarios registrados en PostgreSQL.
4. Registro de eventos de acceso.
5. Almacenamiento y consulta de información histórica.

## Aprendizajes Obtenidos

Durante el desarrollo de este proyecto se fortalecieron conocimientos relacionados con:

* Integración de software y hardware.
* Desarrollo de aplicaciones en C#.
* Diseño y ejecución de consultas SQL.
* Administración de bases de datos PostgreSQL.
* Implementación de lógica de autenticación y control de acceso.
